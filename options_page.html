Command Documentation

General Commands

!jbot: Get the current version of the bot.
Properties
  • Cool Down: 2 mins
  • Access Level: All
Examples
  • Example: !jbot
!JScore x: Get your game score or a another players score.
Properties
  • Player Cool Down: 2 minute
  • Access Level: All
Parameter Set 1
  • (x) Optional: Player name.
Examples
  • Example (Get your lifetime score.): !jscore
  • Example (Get another players score): !jscore @AndrewR74
!Jleader x y: Gets the top x players with the most points.
Properties
  • Global Cool Down: 2 minutes
  • Access Level: All
Parameter Set
  • (x) Optional: Number of top players to display.
  • (y) Optional: Daily or lifetime scores of the top x players.
Examples
  • Example (Get the player with the highest lifetime score): !Jleader
  • Example (Get top 5 lifetime player scores): !Jleader 5
  • Example (Get top 5 daily player scores): !Jleader 5 daily

					<h3>Roulette Commands</h3>
					
					Jroulette - Start a game of roulette.
					Anyone who wants to play has 30 seconds to place bets.

					Access Level: All
					bet <BetName> <[Values]> <Amount> - Place a bet

					Access Level: All
					Mandatory Parameter 1: [ even, odd, red, black, high, low, number, row, rsplit, column, dozen ]
					Optional Parameter 2: integer value. (Only for a Number bet)
					Mandatory Parameter 3: Amount being bet. (Positive integer value)
					Bet Examples
					!bet [Even, Odd, Red, Black, High, Low] <BetAmount>
					!bet number [-1 - 36] <BetAmount>
					-1 = 00
					!bet row [1 - 12] <BetAmount>
					!bet rsplit [1 - 12] [1 - 12] <BetAmount>
					Parameter 1: Row Choice 1
					Parameter 2: Row Choice 2
					Note: Row Choice 2 must be a row next to Row Choice 1
					!bet column [1 - 3] <BetAmount>
					!bet dozen [1 - 3] <BetAmount>
					Note: The 1st dozen, 2nd dozen, 3rd dozen.
					JMoney <PlayerName> - Get a players money

					Access Level: All
					Optional Parameter 1: Players name. Default value is callers name.
					JSpin - Spin the wheel ending the timer.

					Access Level: Broadcaster, Dev, Moderator
					JMoney <Action> <Player> <Amount> - Modify players money.

					Access Level: Broadcaster, Dev
					Parameter 1: [Give, Take, Reset]
					Parameter 2: The players name that the action will be applied to.
					Parameter 3: The amount to taken or given to the player. Exclude parameter for Reset action.
					JSet StartAmount <Amount> - How much money a first time player is given to start playing.

					JSet BetTimeSpan <Seconds> - How long before the bot waits before automatically spinning.

					JSet rTimeSpan <Seconds> - How long players must wait after a game ends before another one can start.

					JSet BustAmount <Amount> - How much money a player that went bankrupt would receive to play again.

					JSet BustModMultiplier <Multiplier> - How much the Bust Amount would be multiplied by when a channel moderator went bankrupt.

					JSet BustSubMultiplier <Multiplier> - How much the Bust Amount would be multiplied by when a channel subscriber went bankrupt.

					JSet ModTimeout <Seconds> - How long a channel moderator has to wait before they can play again after going bankrupt.

					JSet SubTimeout <Seconds> - How long a channel subscriber has to wait before they can play again after going bankrupt.

					JSet PlebTimeOut <Seconds> - How long a channel pleb has to wait before they can play again after going bankrupt.
					</pre>
					
					<h3>JTeam Commands</h3>
					<h4>How To Run </h4>
					<pre>	
					All commands must be follow the !JTeam keyword.

					Note: Team names with spaces must be surrounded with quotation marks. This rule applies to all JTeam commands.

					Example: !JTeam Create "Bobs Team"

					General Commands
					!JTeam <PlayerName>

					Get the current team of a player. The player name is optional and is defaulted to the callers username.
					/all

					List all of the teams on the channel.
					Team Modifiers
					Create <TeamName>

					Create a new team.
					Delete <TeamName>

					Delete an existing team.
					Members <TeamName>

					List all the members of a team.
					<TeamName> Get SubMode

					Get the Sub Mode Only value of a team.
					<TeamName OR /all> Exec <Command Names>

					Execute a team command for a particular team or all channel teams.
					<TeamName> Set SubMode <Boolean>

					Change the Sub Mode Only value of a team.
					<TeamName> Set Name <NewTeamName>

					Change the name of a team.
					Team Members
					<TeamName> Member import

					Import members into a team. Developer Only.
					<TeamName> Member Create @<PlayerName>

					Add a new member to the team. Note: A non subscriber member will be removed from the team if the team is in Sub Mode Only.
					<TeamName> Member Delete @<PlayerName>

					Delete an member from a team.
					<TeamName> Member Mod @<PlayerName>

					Not used.
					<TeamName> Member UnMod @<PlayerName>

					Not used.
					<TeamName> Member AddLeader @<PlayerName>

					Not used.
					<TeamName> Member RmvLeader @<PlayerName>

					Not used.
					Team Commands
					<TeamName OR /all> Command Create <Name> "<Command>"

					Creates a command that can be executed for all team members. The <Name> parameter is used for the execute command. Note: The name can not contain semicolons.

					If the command contains the members name in the command; the <name> tag can be substituted. The members name will automatically be inserted into the command during execution.
					<TeamName OR /all> Command Delete <Name>

					Delete an exising command by it's name.
